# 项目介绍
    简单的逻辑回归数据二分类代码      //自用（CREATED BY JERRY）
    
# 环境依赖
    python (3.10 或者其他)
 
# 目录结构描述
    ├── README.md           // 帮助文档
    
    ├── models              // 存放代码模型文件夹

    │   ├── logistic.py     // 逻辑回归模型

    ├── data                // 存放训练数据文件夹

    │   ├── data.CSV        // 自定义数据集（无意义）

    ├── train.py            // 训练文件，可以修改训练参数，可以训练获得权重文件

    ├── weights.npy         // 通过训练获得的权重文件，不同数据集获得的权重文件不同
 
    ├── detect.py           // 检测文件，完成训练后，可以用新的数据来进行检测
